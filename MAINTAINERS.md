# Maintainers

Nunneri was originated by Suranku and Yamini and is stewarded by the Nunneri Core Team.

## Core Team

- Suranku

## Responsibilities

The core team owns:

- repository direction and release readiness
- licensing and commercial-use decisions
- trademark and brand usage decisions
- CODEOWNERS review and merge approval
- security, compliance, and release-impact escalation
- runtime contract compatibility and adapter release gates
- defensive-publication and citation metadata upkeep

See `GOVERNANCE.md` for the project decision process.

## Sensitive Change Areas

The following changes require Nunneri Core Team review:

- license, commercial-license, trademark, notice, CLA, or ownership text
- `ARCHITECTURE.md`, `DEFENSIVE_PUBLICATION.md`, `CITATION.cff`, and patent-option language
- `SECURITY.md`, authentication, RBAC, tenant isolation, provider-key handling, and approval gates
- release workflows, release packaging, and public homepage changes
