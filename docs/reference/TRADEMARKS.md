## Allowed Uses

You may use the Nunneri name to:

- refer truthfully to the Nunneri project
- describe compatibility with Nunneri assets or runtime contracts
- link to the official repository, docs, or releases
- identify unmodified copies of the Community Edition

## Restricted Uses

Do not use Nunneri marks in a way that:

- implies endorsement, partnership, certification, or sponsorship without written permission
- presents a fork, plugin, hosted service, or commercial package as the official Nunneri project
- uses confusingly similar names, logos, domains, or package names
- removes originator, copyright, or license notices

## Forks and Modified Builds

Forks and modified builds should make their status clear. Use language such as "based on Nunneri" or "compatible with Nunneri" instead of presenting the work as an official Nunneri release.
