Open Source · Run Anywhere · No Lock-In

Nunneri

A mission to open the doors for everyone to follow responsible AI. Enterprise-grade agentic workflows — self-hosted, auditable, and open. Run on a laptop or in any cloud. No vendor lock-in for models or infrastructure.

⬡ Open Graph Studio Developer Setup →

Built on Ollama · LangGraph · PostgreSQL · FastAPI — all open source, all self-hostable.

The Mission

Responsible AI and enterprise agentic workflows should not be the exclusive domain of large companies with large budgets. Nunneri makes them accessible, auditable, and self-hostable — for any team, any size, anywhere.

Run Locally

Fully air-gapped on a laptop with Ollama. No API keys required, no data leaves your machine. Swap in cloud models (Gemini, Claude) when you choose to.

No Model Lock-In

Use any Ollama-supported model today, add GEMINI_API_KEY or ANTHROPIC_API_KEY for cloud models. One prefix change — gemini:gemini-2.0-flash — is all it takes.

No Infrastructure Lock-In

Standard PostgreSQL for state. Standard OIDC (Keycloak, Okta, Azure AD) for auth. Standard FastAPI for the server. Deploy it yourself, on any host.

Responsible by Design

Human approval gates in every workflow. Full run audit trail with per-node state in the database. Role-based access across org → team → project. Error detail stored, not lost.

Provider-Agnostic Assets

One source of truth generates context for Claude, Codex, Gemini, and open-source frameworks. No duplicated prompts, no provider drift.

Open Core, Open Roadmap

The core is open source. Issues and contributions through GitHub. The roadmap belongs to the community.

Graph Studio

Graph Studio is the operational interface for running agents. It is separate from this documentation portal — agents here are provider context files; in Graph Studio they are executable LangGraph workflows with live DAG visualisation, thread history, and per-node configuration.

⬡ Open Graph Studio

Select an agent or command, attach a project path, run it, and watch the graph execute node by node with live token streaming.

Launch Graph Studio ↗

Requires uvicorn api.main:app and Ollama running locally.

Thread History

Every run is persisted to PostgreSQL. Click any thread to restore the graph view with node completion states, the full run output, and the error detail if it failed.

Node Configuration

Click any node in the graph to set a per-node system prompt, classification labels, and routing rules — stored in the database and applied on every run.

LLM Providers

Runs on local Ollama by default. Add GEMINI_API_KEY or ANTHROPIC_API_KEY to enable cloud models (gemini:gemini-2.0-flash, claude:claude-sonnet-4-6, …).
